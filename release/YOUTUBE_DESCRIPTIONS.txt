TITLE:
JVM AI Integration Done Right: FastAI v0.1.4 Released! 🚀

DESCRIPTION:
FastAI v0.1.4 is now available! This release focuses on performance optimizations, memory footprint reduction, and cleaning up runtime artifact outputs. 

FastAI is a zero-dependency, lightweight, unified AI client interface for Java that connects seamlessly to local (Ollama, Llama.cpp) and cloud models (Gemini, Claude, OpenAI) in under 100ms.

KEY HIGHLIGHTS:
- [Performance]: Eliminated diagnostic disk logging overhead to achieve maximum execution speed during live token streaming.
- [Ecosystem Bumps]: Updated internal model pipelines to fully support FastAIRag's native JNI operations.
- [Clean Outputs]: Streamlined console standard error redirections for cleaner terminal integrations.

RESULTS:
- Connection time: <100ms
- Memory Overhead: Negligible
- Dependencies: 0
